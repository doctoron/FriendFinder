### Setup
Running Locally
Make sure you have Node.js and the Heroku Toolbelt installed.
git clone git@github.com:heroku/node-js-sample.git # or clone your own fork
cd node-js-sample
npm install
npm start
Your app should now be running on localhost:8080.
